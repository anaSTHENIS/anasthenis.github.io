About

The D&D Event Announcer integrates tightly with Discord for real-time announcements and event management, helping Dungeon Masters and players coordinate their campaigns effectively.

Features

  • ✅ Create D&D event announcements with custom imagery
  • ✅ Automatic event scheduling and reminders
  • ✅ Multi-language support (9 languages)
  • ✅ Easy setup with slash commands
  • ✅ Event management (list, add, delete)

Quick Start

  1. Add the bot to your Discord server
  2. Use /setchannel to set announcement channel
  3. Use /addevent to create your first event
  4. Done! The bot will automatically post announcements

Commands

/addevent - Create a new D&D event announcement

/listevents - View all scheduled events

/deleteevent - Remove an event

/setchannel - Set the channel for event announcements

/setlanguage - Change bot language
